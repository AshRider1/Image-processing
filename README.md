# Image-processing

## Dataset

This project uses the BDD100K dataset from Kaggle:

[https://www.kaggle.com/datasets/marquis03/bdd100k](https://www.kaggle.com/datasets/marquis03/bdd100k)

The dataset is not included in this repository because it is too large.

Download the dataset from Kaggle, unzip it, and place the files in the following structure:

```text
Image-processing/
├── dataset/
│   ├── train/
│   ├── val/
│   └── test/
├── ...
└── requirements
```
